# Клиент-серверное приложение SmartBridge для поиска работы и подбора персонала.

Поддерживает работу для пользователей с двумя ролями: Соискатель и HR-специалист. Общими для них является интерфейс авторизации, регистрации, сброса пароля и настроек. Соискатель дополнительно может создавать новые анкеты-резюме с описание своих требований к работе мечты, просматривать свои резюме. HR имеет возможность искать резюме при помощи фильтров, добавлять их в избранное, переходить в мессенджеры, создавать шаблоны сообщений для соискателей.
